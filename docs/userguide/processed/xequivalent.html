McIDAS-X Commands in McIDAS-V

Listed Below are several common McIDAS-X commands, followed by a brief description of the corresponding method used in McIDAS-V.

DATALOC
  • ADD - Enter server and dataset information in the chooser, or use the ADD ADDE Servers... button in the ADDE Servers tab of the User Preferences window.
  • LIST - Select server in chooser and use Dataset pull down menu to view available datasets.
IMG*
  • IMGDISP - Use the Satellite Imagery chooser to select dataset, add the data, and load the image(s).
  • IMGFILT - Once the data is loaded in the Field Selector, select Formulas -> Image Filters.
  • IMGLIST - Use the Image Type pull down menu and the Absolute Times tab to view the available list of times from the server. Channels and image size information is listed below.
  • IMGOPER - Create or use formulas or Jython methods.
  • IMGPLOT - Transect/Probe displays.
  • IMGPROBE - Click the middle mouse button or use Probe display type.
  • IMGREMAP - Use the Projections menu to create or change projections.

EU
  • LIST - Right-click on the color table in the Legend or Layer Controls to get a list of available color tables.
  • MAKE - Use the Tools -> Color Table editor.
  • REST - Right-click on the color table in the Legend or Layer Controls and select a new enhancement.
ALT
  • A, B, L - Use the VCR commands above the 3D display to control the looping.
  • D - Use the middle mouse button click option to probe the data in the main display window.
  • K, W - Use the layer checkboxes in the Legend to control the visibility status of images, maps, and other layers in the 3D display.
  • O - Use layer checkboxes in the Legend to control visibility, or use Bring View to Front option to reorganize layers.
  • Use Ctrl+N to advance to the next tab, Ctrl+P to go to the previous tab, or click on the tab you wish to view.
GRD*
  • GRDDISP - Use the Gridded Data Chooser to load data from a THREDDS server or from a local grib file and to select the display option for gridded data.
  • GRDIMG - Display gridded data as a Color-Shaded Plan View.
SFC*
  • SFC* - Use the Point Observations -> Plot/Contour chooser to select surface data.
  • SFCCON - Use Gridded Fields to display contours in a Plan View display.
  • SFCLIST - Display Point Data as a Point Data Plot, and use the Chart tab in the Layer Controls to view the fields for a specific station.
  • SFCPLOT - Display Point Data as a Point Data Plot, and choose Layout Model for how to plot the point data. Use the Tools -> Station Model Template editor to create/edit a new station model.
  • SFCRPT - Weather Text Products Display (Tools -> Text Data -> Weather Text Products (from server). Select Observed Data -> Sounding Data Product, and the station in the Locations tab).
RAOB*
  • RAOB* - Use the Point Observations -> Soundings chooser to select upper air data.
  • RAOBCON - Use the Gridded Fields to display contours in a Plan View Display.
  • RAOBPLOT - Display Point Data as a Point Data Plot, and choose Layout Model for how to plot the point data. Use the Tools -> Station Model Template editor to create/edit a new station model.
  • RAOBPRT - Weather Text Products Display (Tools -> Text Data -> Weather Text Products (from server). Select Observed Data -> Sounding Data Product, and the station in the Locations tab.
Other
  • DSINFO - Select the chooser with the data you want to view (Image, Point, etc.), connect to the server and view "Data" type list to see available descriptors.
  • COMBINE - Formulas: Imagery -> RGB, or create your own formula/Jython method.
  • PTDISP - Use the Point Observations-> Plot/Contour chooser to select data, and display Point Data as a Point Data Plot and choose your Layout Model for how to plot your point data. Use the Tools -> Station Model Template editor to create/edit a new station model.
  • Scheduler (SKED, SKE, SKL, SKU) - Set up polling options in data sources (right-click on the Data Source in the Field Selector and select Properties), or run background scripts through cron.
  • WWDISP - Shape files from the General Catalogs chooser, or use the McIDAS-X Bridge.
  • ZA - Text option in the Drawing Controls (Display -> Draw Freely).
  • ZLM, CW - Shape options in the Drawing Controls (Display -> Draw Freely).
