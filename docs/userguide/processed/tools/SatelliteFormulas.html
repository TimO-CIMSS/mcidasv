Description of Formulas - Satellite

The following describes each formula included in the Miscellaneous Formulas section of McIDAS-V. These formulas are designed to work with SEVIRI MSG satellite data, although they are also applicable for equivalent channels of other imagers.

SEVIRI RGB

  • AMCP - Displays all-day (24 hour) cloud microphysics data. A Field Selector window will appear when you click Create Display. In this window you select three fields and press OK. The formula will compute the fields to display AMCP data in the Main Display window. The display will be in units of temperature in degrees K.
  • AIRM - Displays airmass data. A Field Selector window will appear when you click Create Display. In this window you select four fields and press OK. The formula will compute the fields to display AIRM data in the Main Display window. The display will be in units of temperature in degrees K.
  • DUST - Displays dust data. A Field Selector window will appear when you click Create Display. In this window, you select three fields and press OK. The formula will compute the fields to display DUST data in the Main Display window. The display will be in units of temperature in degrees K.
  • NCMP - Displays night cloud microphysics data. A Field Selector window will appear when you click Create Display. In this window you select three fields and press OK. The formula will compute the fields to display NCMP data in the Main Display window. The display will be in units of temperature in degrees K.
  • NCOL gamma - Displays natural colors data. A Select Input window will appear when you click Create Display. In this window, you enter a gamma value before hitting OK. The gamma value enhances the contrast between the low and high reflectivities. After you hit OK, a Field Selector window will appear. In this window you select two fields and press OK. The formula will compute the fields along with your gamma value to display NCOL data in the Main Display window. The display will be in units of reflectivity.
  • NCOL linear - Displays natural colors data. A Select Input window will appear when you click Create Display. In this window, you enter minimum and maximum reflectivity values before hitting OK. After you hit OK, a Field Selector window will appear. In this window you select one field and press OK. The formula will compute this field along with your minimum and maximum reflectivity values to display NOCL data in the Main Display window. The display will be in units of reflectivity.
  • SCON - Displays severe convection data. A Field Selector window will appear when you click Create Display. In this window you select six fields and press OK. The formula will compute the fields to display SCON data in the Main Display window. The display will be in units of temperature in degrees K and in units of reflectivity.
  • VOLC - Displays sulfur dioxide and volcanic ash data. A Field Selector window will appear when you click Create Display. In this window you select three fields and press OK. The formula will compute the fields to display VOLC data in the Main Display window. The display will be in units of temperature in degrees K.

SEVIRI RGB Manual band selection

  • Airmass: 05-06, 08-09, 05i (AIRM) - Displays airmass data. A Field Selector window will appear when you click Create Display. This window includes four fields, labeled WV62 (Water Vapor 6.2um), WV73 (Water Vapor 7.3um), IR97 (Infrared 9.7um), and IR108 (Infrared 10.8um). Once you select these fields and click OK, the formula will compute these fields to display AIRM data in the Main Display window. The display will be in units of temperature in degrees K.
  • All-day cloud microphysics: 10-09, 09-07, 09 (AMCP) - Displays all-day (24 hour) cloud microphysics data. A Field Selector window will appear when you click Create Display. This window includes three fields, labeled IR87 (Infrared 8.7um), IR108 (Infrared 10.8um), IR120 (Infrared 12.0um). Once you select these fields and click OK, the formula will compute these fields to display AMCP data in the Main Display window. The display will be in units of temperature in degrees K.
  • Dust: 10-09, 09-07, 09 (DUST) - Displays dust data. A Field Selector window will appear when you click Create Display. This window includes three fields, labeled IR87 (Infrared 8.7um), IR108 (Infrared 10.8um), IR120 (Infrared 12.0um). Once you select these fields and click OK, the formula will compute these fields to display DUST data in the Main Display window. The display will be in units of temperature in degrees K.
  • Natural colors: 03, 02, 01, gamma - Displays natural colors data. A Select Input window will appear when you click Create Display. In this window, you enter a gamma value before hitting OK. The gamma value enhances the contrast between the low and high reflectivities. After you hit OK, a Field Selector window will appear. This window includes three fields, labeled VIS06 (Visible 0.6um), Vis08 (Vis 0.8um), NIR16 (Near-Infrared 1.6um). Once you select these fields and click OK, the formula will compute these fields to display Natural colors (gamma) data in the Main Display window. The display will be in units of reflectivity.
  • Natural colors: 03, 02, 01, minrefl, maxrefl - Displays natural colors data. A Select Input window will appear when you click Create Display. In this window, you enter minimum and maximum reflectivity values before hitting OK. After you hit OK, a Field Selector window will appear. This window includes three fields, labeled VIS06 (Visible 0.6um), Vis08 (Vis 0.8um), NIR16 (Near-Infrared 1.6um). Once you select these fields and click OK, the formula will compute these fields to display Natural colors minrefl, maxrefl) data in the Main Display window. The display will be in units of reflectivity.
  • Night cloud microphysics: 10-09, 09-04, 09 (NCMP) - Displays natural colors data. A Field Selector window will appear when you click Create Display. This window includes three fields, labeled IR39 (Infrared 3.9um), IR108 (Infrared 10.8um), IR120 (Infrared 12.0um). Once you select these fields and click OK, the formula will compute these fields to display NCMP data in the Main Display window. The display will be in units of temperature in degrees K.
  • SO2 & ash: 10-09, 09-07, 09 (VOLC) - Displays severe convection data. A Field Selector window will appear when you click Create Display. This window includes three fields, labeled IR87 (Infrared 8.7um), IR108 (Infrared 10.8um), IR120 (Infrared 12.0um). Once you select these fields and click OK, the formula will compute these fields to display VOLC data in the Main Display window. The display will be in units of temperature in degrees K.
  • Severe convection: 05-06, 04-09, 03-01 (SCON) - Displays severe convection data. A Field Selector window will appear when you click Create Display. This window includes six fields, labeled ch01, ch03, ch04, ch05, ch06, and ch09. Once you select these fields and click OK, the formula will compute these fields to display SCON data in the Main Display window. The display will be in units of temperature in degrees K and in units of reflectivity.

SEVIRI Differences

  • OTOP (IR10.8 - WV6.3) - Displays overshooting tops data. A Field Selector window will appear when you click Create Display. This window includes two fields, D4 (Infrared 10.8um) and D1 (Water Vapor 6.3um). Once you select these fields and click OK, the formula will compute these fields to display OTOP data in the Main Display window.
