6.12 Support Request Form
Clicking on the main menu bar choice Help->Support Request Form... shows a form to post a support request to the Unidata Program Center (UPC). This form can also be brought up from the Error Dialog.

Image 1: Support Request Form
If you encounter a problem or have a comment to share, you can use this form to do so.

By checking on the Include Current State as Bundle button the current state of McIDAS-V will be submitted along with your request. You can also add attachments to your submission.
