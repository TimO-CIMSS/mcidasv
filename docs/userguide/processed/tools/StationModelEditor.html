Layout Model Editor

The Layout Model Editor composes the layout and style used to plot data on a map from observations at discrete points (e.g., weather observations). This tool is used to edit existing models or to create new models. The Editor is opened with the Tools->Station Model Template menu in the Main Display Window.

  • Pop-up Menu
  • Properties Dialog - Display Tab
  • Properties Dialog - Color By Tab
  • Properties Dialog - Scale Size Tab
  • Properties Dialog - Rotate Tab
  • Menus

Image 1: Layout Model Editor

You can create your own layout model from scratch by utilizing the File->New menu option, or you can select an existing layout model with the Layout Models menu. The display includes a horizontal and vertical line, which acts as a reference point for the location of your layout model. The different shapes and features that you can add to the layout model are in the left column. They can be added to the layout model by left-clicking the item and then clicking where you want to place it. The individual shapes in the layout model can be moved by dragging them with the mouse.

Each feature of the layout model can be resized by dragging the black squares along the outside of the item. Each item in the model also includes a red "Alignment Point', which acts to locate the individual components of the display with respect to the center point of the observation. You can select multiple items to align at once by holding down Shift and clicking on various items. The location of the Alignment Point to the item can be changed via the Alignment Point option in the Pop-up Menu.

To save a station model you have composed, use the File->Save or File->Save As... menu items. When you save a station model any point data displays that are using it will be updated. To remove a model, use the File->Remove menu item. You cannot remove a model supplied with McIDAS-V, but you can remove local models.

Pop-up Menu

There are many options that you can control for each individual component of the layout model. This menu is seen by right-clicking on an item.

  • Set Color - Sets the color of the selected item.
  • To Front - Brings the selected item to the front of others that may overlap it.
  • To Back - Brings the selected item to the back of others that may overlap it.
  • Delete - Removes the selected item from the layout model display.
  • Hide - Grays out the component in the layout model, still leaving the object in the model, but not displaying it in the Main Display Window.
  • Alignment Point - Sets the location of the Alignment Point with respect to the item. For example, NW would place the Alignment Point to the upper left of the item.
  • Center - Centers the display in both the horizontal and vertical, just in the horizontal, or just in the vertical.
  • Properties... - Opens the Properties Dialog Window described below.
The Properties window consists of four tabs: Display, Color By, Scale Size, and Rotate.
Properties Dialog - Display Tab

The Display tab allows you to assign which observation variable is associated with that item. Numerical values should appear in a "Value" item, text in a "Text" item, etc.


Image 2: Properties Dialog - Display

Properties
  • Value parameter - Represents what the parameter is displaying. This can be a single parameter, such as pressure, or it can be an equation. This is possible because the parameter identifers in the layout model can hold embedded Jython code, and any parameter that begins with "=" is treated as a Jython expression. For example, you can type "=T-Td" for dew point depression. You can also format this as "='Td:'+str(T-Td)". If you want to format the result of the calculation using the format specifier from the properties dialog, then enter "='Td:'+format(T-Td).
  • Unit - Determines the unit that the parameter will use in its display.
  • Format - Represents how the numerical value will be displayed. This can affect rounding, an example of how each format will appear is written in the 'ex:' next to the Format field.
  • Font - Sets the font that the item will be written in when displayed in the Main Display Window.
  • Size - Sets the size that the item will be written in when displayed in the Main Display Window.
  • Shown - Determines if the item will be shown as part of the layout model.
  • Foreground Color - Represents the color of the text.
  • Background Color - Sets a background color for the text if desired. A through this field is representative of a transparent background.
  • Scale Size By - Scales the size of the item with respect to other items in the layout model.
Properties Dialog - Color By Tab

The Color By tab consists of two tabs within itself: Map Value Into Color Table, and Color From Parameter. This tab allows you to determine how you want to color your layout model.

The Map Value Into Color Table tab allows you to color the parameter based off of its numerical value in relation to the set data range and color table for the parameter.


Image 3: Properties Dialog - Color By

Properties
  • Map Value of - Sets the parameter you want to work with.
  • Data Range - Sets the numerical range of the parameter listed in the Map Value field. This represents the end points for the color table.
  • Unit - Sets the unit of the parameter listed in the Map Value field.
  • Color Table - Sets the color table of the parameter to be one of the already existing tables.

For example, in the above image the Temperature parameter has been specified with data range -90-45 celsius. For each display McIDAS-V will find the Temperature value (e.g., 15 celsius), convert it, if necessary to the specified unit, find its where is lays within the range and use the corresponding color from the color table.

The Color From Parameter tab allows you to specify parameter, a set of (regular expression) patterns and a corresponding color. If the text value of the parameter matches one of the patterns then the given color is used to color the shape.


Image 4: Properties Dialog - Color From

Properties
  • Get Color From - Selects a parameter or set of patterns that correspond to a color.
  • Parameter Value - Represents a numerical, fixed value that will be colored by the color selected for each value. Pressing the button will give you two more Parameter Value fields once all other fields are filled.
Properties Dialog - Scale Size Tab

This allows you to specify a parameter to scale the shape by. For each point display, McIDAS-V will find the value of the parameter, determine where that value lies within the data range, and find the corresponding scaling from the scale factor range.


Image 5: Properties Dialog - Scale Size By

Properties
  • Scale By Parameter - Sets the parameter you want to work with.
  • Data Range - Sets the numerical range of the parameter listed in the Map Value field.
  • Unit - Sets the unit of the parameter listed in the Scale By Parameter field.
  • Scale By Factor - Sets the range for which the scaling factor will be applied to the data.
Properties Dialog - Rotate Tab

The Rotate About Axis tab works like the scale and color by facilities. Note, the rotation is about the Z axis in X/Y/Z space. It is not a rotation in Lat/Lon space. Only the Rotate About Z Axis tab is shown as all of the Axes have identical fields.


Image 6: Properties Dialog - Rotate By

Properties
  • Rotate By Parameter - Sets the parameter you want to work with.
  • Data Range - Sets the numerical range of the parameter listed in the Map Value field.
  • Unit - Sets the unit of the parameter listed in the Scale By Parameter field.
  • Degrees - Sets the range of degrees for which the data will rote about each respective axis.
Menus

Many of the menu items seen utilizing this display are standard options that can be found in the Menus section of the Layer Controls Page. However, there are several options unique to this display.

The File menu has these unique options:

  • New - Opens a New Layout Model window that prompts for the name of the Layout Model you are going to create.
  • Save - Saves the Layout Model so that it can be used in McIDAS-V.
  • Save As... - Saves the Layout Model so that it can be used in McIDAS-V.
  • Rename... - Opens a Rename Layout Model window that prompts for the name you wish to change the Layout Model to.
  • Remove - Removes the Layout Model that you are viewing.
  • Import... - Allows you to import a Layout Model as an IDV station model file (*.ism).
  • Export... - Allows you to export a Layout Model as an IDV station model file (*.ism).
  • Close... - Closes the Layout Model Editor.

The Edit menu has these unique options:

  • Symbols - Lists the different symbols that you have in your Layout Model. Clicking on the symbol will bring up the Properties Dialog window for that symbol, allowing you to make several modifications to it.
  • Set properties on selected - Opens a Properties Dialog window for the symbol you have selected in the Layout Model Editor. This allows you to change the range, set the unit, and set a color table for the symbol.
  • Cut - Removes the selected symbol from the Layout Model.
  • Copy - Copies the selected symbol from the Layout Model.
  • Paste - Pastes a symbol into the Layout Model.
  • Select All - Selects each individual symbol in the Layout Model such that you can perform a cut/copy/paste to all of the symbols at once.
  • To Front - Places the selected symbol in front of the others.
  • To Back - Places the selected symbol behind the others.

The View menu has these unique options:

  • Show Alignment Points - Displays a red square representing where the symbol will be centered.
  • Zoom in - Zooms in on the Layout Model.
  • Zoom out - Zooms out on the Layout Model.
  • Zoom rest - Reverts back to original zoom level.
  • Grid - Adds horizontal/vertical grids to the Layout Model. Increase makes the grid size larger, and decrease makes the grid size smaller.
  • Black background - Makes the background in the Layout Model Editor Black.
  • White Background - Makes the background in the Layout Model Editor White.

The Layout Models menu has these unique options:

  • Location - Changes the Layout Model to display the name of the locations of the Layout Models in the Main Display Window.
  • Label - Changes the layout Model to be a label when displayed in the Main Display Window. You can make the Label be anything.
  • Selected - Removes any symbols in the Layout Model, leaving just the Alignment Point.
  • Observations/Tracks/Grids/etc. - Allows you change the Layout Model to be one of the saved symbols/parameters.
