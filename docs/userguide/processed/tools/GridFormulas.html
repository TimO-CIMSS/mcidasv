Description of Formulas - Grids

The following describes each formula included in the Grids Formulas section of McIDAS-V.

  • Time Steps
  • Ensemble
  • Miscellaneous
Time Steps

Average Time steps - Plots one map representing the average value of the specified parameter at any one point on the map during the time sequence selected.

Difference from initial timestep (D(T)-D(T)-D(0)) - Plots a sequence of images. Each image maps the difference in the numerical value of the parameter from the initial values of the first frame. This formula sets the value of each time step N as:

 

D(T) = D(T) - D(0)

Max over time steps - Plots one map representing the highest value of the specified parameter at any point on the map during the time sequence selected.

Merge time sequences - ????????????

Min over time steps - Plots one map representing the lowest value of the specified parameter at any one point on the map during the time sequence selected.

Sum over time steps - Plots one map representing the total value of the parameter at any one point on the map during the time sequence selected.

Time step difference (D(T)=D(T)-D(T-1)) - Plots a sequence of images. Each frame plots the value of the specified parameter at that time, and subtracts from it the value of the same parameter at the previous time in the loop.

Time step sum (D(T)=D(T)+D(T-1)) - Plots a sequence of images. Each frame plots the value of the specified parameter at that time, and adds to it the value of the same parameter at the previous time step in the loop.

Miscellaneous

Define a grid diagnostic (gddiag) - Allows for defining a grid diagnostic to perform.

Input parameters:

gddiag

grid diagnostic. For more information see the Grid Diagnostics section of the Jython Library.

Find latitude of grid points (feldimpl) - Obtains the latitude coordinate from the grid at each location and plots the latitude at each point on the map.

Find longitude of grid points (fieldimpl) - Obtains the longitude coordinate from the grid at each location and plots the longitude at each point on the map.

Horizontal Advection (from %N1% & %N2%) (scalar parameter, U Component, V Component)

This formula calculates horizontal advection from U and V components.

Input parameters:

scalar

scalar parameter

Horizontal Flux Divergence (from %N1% & %N2%) (scalar parameter, U Component, V Component) - Calculates horizontal flux divergence from U and V components.

Input parameters:

scalar

scalar parameter

Layer Difference (Top of Layer, Bottom of Layer) - Calculates a layer difference between a field at two different levels.

Layer Wind Shear (top, bottom) - Calculates the wind shear between two user-defined layers.

Input parameters:

top

top layer (default=500)

bottom

bottom layer (default=850)

Make flow vectors from grid relative u and v data (u, v) - Calculates true wind vectors based off of relative u and v data after user inputs the field and desired levels. You can utilize 2D or 3D data.

Make flow vectors from u and v data (u, v) - Creates flow vectors based off of the 2D u and v parameters entered by the user.

Make flow vectors from u, v, and w data (u, v, w) - Creates flow vectors based off of the 3D u and v parameters entered by the user. These flow vectors are then scaled based off the user-entered w parameter.

Make slice 2D (slice) - Creates a 2D slice from a 3D display at a single level.

One value colored by another (Data field, Color field) - Creates an image of the data chosen in the Data field, and colors the data with the color scheme of the Color field parameter.

Resample grid - Displays gridded data on a new domain.

Simple time difference in one parameter (Time 1, Time 2) - Creates an image representing the numerical difference in the value of the chosen parameter between the two time steps selected by the user. Time 1 is the initial time step, and Time 2 is the second time step. The difference in the parameter is done by Time 2 - Time 1.

Input parameters:

Time 1

first time (default=1)

Time 2

second time (default=0)

Slice at Altitude (Altitude, Unit) - Extracts a 2D horizontal slice from a 3D grid at the given altitude and maps an image.

Input parameters:

altitude

altitude to slice

unit

specify a unit. It must be compatible with meters (ft, fathoms, etc.)

Slice at Level (Level) - Extracts a 2D horizontal slice from a 3D grid at a level.

Input parameter:

level

level to slice, must be appropriate for the grid

Speed from 2 components (Field x, Field y) - Calculates speed from the user-chosen x and y components and displays an image.

Wind Shear Vectors (top, bottom) - Calculates the u and v layer difference and returns a map of vectors.

Input parameters:

top

top layer (default=500)

bottom

bottom layer (default=850)
