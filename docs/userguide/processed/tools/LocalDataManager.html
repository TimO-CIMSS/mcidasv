Local ADDE Data Manager

The Local ADDE Data manager is opened through the Tools menu with Tools->Local ADDE Data menu and allows you to add and manage local image files on your machine.


Image 1: Local Data Manager

To add local data entries, select File->New Entry from the menu. The following parameters are required for setting up a local server entry:

Dataset (Group) Enter in a dataset name for the local server entry. The dataset name is limited to eight characters or less and cannot be entirely numeric.

Image Type (Descriptor) Enter in an image type name for the new local server entry. This will appear under the Image Type pull down menu in the Satellite Imagery chooser. The image type name is limited to twelve characters or less.

Format Select an image format.

The available local image servers in McIDAS-V:

  • AMSR-E Level 1b
  • AMSR-E Rain Product
  • LRIT
  • McIDAS AREA
  • Meteosat OpenMTP
  • Metop AVHRR Level 1b
  • MODIS L1b MOD02 (MODIS Level 1b)
  • MODIS L2 MOD04 (MODIS Level 2 Aerosol)
  • MODIS L2 MOD06 (MODIS Level 2 Cloud Top Properties)
  • MODIS L2 MOD07 (MODIS Level 2 Atmospheric Profile)
  • MODIS L2 MOD35 (MODIS Level 2 Cloud Mask)
  • MODIS L3 MODR (MODIS Level 2 Corrected Reflectance)
  • MSG HRIT
  • MTSAT HRIT
  • NOAA AVHRR Level 1b
  • SSMI (TeraScan netCDF)
  • TRMM (TeraScan netCDF)

Directory Select the location of the local files using the pop up navigation box. At this time, only a directory can be selected. An option to add files by file mask will be added in the future.

To edit an entry, double click or right click on the table row and select "Edit Entry." To delete an entry, right click on the table row and select "Delete Entry." The port that the local server is listening on is listed at the bottom of the local data manager window.

If the local server is not running or needs to be restarted, click the Restart icon.
