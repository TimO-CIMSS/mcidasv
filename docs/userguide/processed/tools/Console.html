Message Console

Clicking on the main menu bar choice Help->Show console shows a console window which has error messages and other text output from McIDAS-V. Error messages created before you open the console are shown, as well as later ones.

Properties
  • - Clears the Console of all messages.
  • - Allows you to save your Console messages in the Log files (*.log) format.

The console is useful if you encounter an error as you can save the console message to a file and include it in email to McIDAS-V Support at SSEC or attach it in the Support Request Form.
