User Preferences - ADDE Servers

The User Preferences window is accessed through the Edit->Preferences menu. The ADDE Servers tab allows the user to add and manage a list of active ADDE servers separated by data type. The window is also accessible by clicking the Manage servers button () next to the Server name of the applicable choosers in the "Data Sources" tab of the Data Explorer. The image below represents the default settings for the for the ADDE Servers tab. Everything is checked on except 'NAV' and 'UNKNOWN'.


Image 1: ADDE Server Preferences (Default)

Properties

Use all ADDE entries - Selecting this bubble will select all of the ADDE entries.

Use selected ADDE entries - Selecting this bubble allows the user to select which ADDE entries desired.

All on - Selecting this box will select all of the ADDE entries.

All off - Selecting this box will unselect all of the ADDE entries.

Add ADDE Servers... - Selecting this box will open the ADDE Data Manager where the user can add a remote server or a local server.

Import Servers... - Selecting this box will open the ADDE Data Manager where the user can add a remote server or a local server.
