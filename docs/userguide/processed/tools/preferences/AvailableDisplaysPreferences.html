User Preferences - Available Displays

The User Preferences window is accessed in the Main Display Window through the Edit->Preferences menu. The Available Displays tab allows the user to specify what types of displays to use. The chosen displays will be included in the Displays list in the Field Selector when you attempt to create an image. Choosing what displays you want to appear in the Field Selector will limit the number of displays in the list, making it easier to create your desired image. The image below represents the default settings for the Available Displays tab. Every option is checked on by default.


Image 1: Available Displays Preferences (Default)

Properties
  • Use all displays - Selects all of the displays.
  • Use selected displays - Uses only selected displays.
  • - Selects all of the displays.
  • - Unselects all of the displays.
