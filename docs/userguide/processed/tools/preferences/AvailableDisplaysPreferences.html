User Preferences - Available Displays

The Available Displays tab of the User Preferences window (which is accessed through the Edit->Preferences menu) allows you to specify what types of displays to use. The image below represents the default settings for the Available Displays tab in the User Preferences window. Every option is checked on by default.


Image 1: Available Displays Preferences (Default)

You can either select to "Use all displays" or "Use selected displays", which provides the user the freedom to choose which displays should be used. The list of displays is hierarchical. You can turn on or off all of the displays, specific groupings or individual displays.
