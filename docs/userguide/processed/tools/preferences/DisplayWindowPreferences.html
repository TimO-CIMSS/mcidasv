User Preferences - Display Window

The User Preferences window is accessed through the Edit->Preferences menu. The Display Window tab allows the user to set preferences related to the main 3D window. The image below represents the default settings for the Display Window tab.


Image 1: Display Window Preferences (Default)

 

Navigation Toolbars

Show Globe Background - this, when checked, displays a solid globe background.

Show Earth Navigation Panel - this, when checked, shows a graphical navigation panel.

Show Viewpoint Toolbar - this, when checked, enables the Viewpoint toolbar.

Show Zoom/Pan Toolbar - this, when checked, enable the Zoom/Pan toolbar.

Show Undo/Redo Toolbar - this, when checked enables the Undo/Redo toolbar.

Legends

Show Side Legend - this, when checked, shows the side display legend.

Show Bottom Legend - this, when checked, shows the bottom display legend.

Panel Configuration

Show Wireframe Box - this, when checked, toggles the visibility of a 3D wireframe box around the data. This box helps provide orientation in 3D views.

Show Cursor Readout - this, when checked, displays a label of latitude, longitude, and altitude of the cursor position over the display, shown in the lower border of the view window.

Clip View At Box - this, when checked, allows you to elide from view all display elements outside the wireframe box. This is helpful to remove map lines outside the area of data.

Show Layer List in Panel - this, when checked, shows a listing of the titles and times of the layer controls in the main view window.

Show Times In Panel - this, when checked, will turn on the label shown inside the main display that shows the current animation time.

Show Map Display Scales - this, when checked, will show a vertical scale in the map displays.

Show Transect Display Scales - this, when checked, will show a vertical scale in the transect displays.

Show "Please Wait" Message - this, when checked, will display the "Please Wait" message in the display.

Reset Projection With New Data - this, when checked, will cause each new display made in McIDAS-V to snap the whole view window to the native projection of the data. This can be undesirable if you are overlaying several kinds of some of which have differing projections.

Color Scheme

Globe Background - this is the color of the globe's background.

Background - this is the color of the background behind the image being displayed.

Foreground - this is the color of the wireframe box in the foreground.

Selected Panel - this is the color in the frame around the selected panel. This is the panel that changes will be made to if you have multiple tabs open.

Note: The color of the maps is controlled by the individual map control. The selected panel color changes the color used to highlight the current panel.

Layer List Properties

Font - this is the text at the bottom of the view screen denoting what is being viewed. Using the drop-down menus, the user can define the font, style, and size of text.

Color - this is the color of the text at the bottom of the view screen.

Default Projection
Default Projection - this drop-down menu allows the user to choose which map projection McIDAS-V will default to.
