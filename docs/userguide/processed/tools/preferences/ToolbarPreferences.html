User Preferences - Toolbar Options

The Toolbar Options tab of the User Preferences window (which is accessed through the Edit-> Preferences menu). This tab allows you to define what action/icons are displayed in the toolbar. The list on the left shows all of the available actions. The list on the right shows the actions which make up the toolbar. The image below shows the default settings for the Toolbar Options.


Image 1: Toolbar Preferences (Defaults)

You can add/remove actions to/from the toolbar list with the Add >/< Remove buttons. The Add space button adds a space between toolbar entries. The Reload Original resets the toolbar list to the one before any changes have been made. The Export Selected to Plugin button will open the Plugin Creator window, with the selected action included under the resources tab. The Export Selected to Menu Plugin will bring up the Export to Menu window, where the user can enter in a Menu Name, and select which menu to add the action under.
