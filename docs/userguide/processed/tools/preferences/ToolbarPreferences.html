User Preferences - Toolbar Options

Image 1: Toolbar Preferences
The Toolbar Options tab allows you to define what action/icons are displayed in the toolbar. The list on the left shows all of the available actions. The list on the right shows the actions which make up the toolbar. You can add/remove actions to/from the toolbar list. The Add space button adds a space between toolbar entries. The Reload Original resets the toolbar list to the one before any changes have been made.
