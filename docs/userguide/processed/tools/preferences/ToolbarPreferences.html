User Preferences - Toolbar Options

The User Preferences window is accessed through the Edit->Preferences menu. The Toolbar Options tab allows the user to define what actions/icons are displayed in the toolbar. The list on the left shows all of the available actions. The list on the right shows the actions that make up the toolbar. The image below represents the default settings for the Toolbar Options tab.


Image 1: Toolbar Preferences (Default)

Properties
  • Add > - Adds selected item on left column to the right column, bringing this item to the toolbar.
  • < Remove - Takes selected from right column to the left column, removing this item from the toolbar.
  • Add space - Adds a space between the icons on your toolbar.
  • Reload Original - Undos any user changes to the Toolbar Options tab, returning the Actions and Toolbar columns to the default settings of McIDAS-V.
  • Export Selected to Plugin - Brings up the Plugin Creator window with the selected action included under the resources tab.
  • Export Selected to Menu Plugin - Brings up the Export to Menu window where the user can enter in a Menu Name and select which menu to add the action under.
  • - Moves the selected action in the toolbar column up in the list, changing the location of the action in the toolbar.
  • - Moves the selected action in the toolbar column down in the list, changing the location of the action in the toolbar.
