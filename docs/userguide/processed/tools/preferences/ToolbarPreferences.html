User Preferences - Toolbar Options

The User Preferences window is accessed through the Edit->Preferences menu. The Toolbar Options tab allows the user to define what action/icons are displayed in the toolbar. The list on the left shows all of the available actions. The list on the right shows the actions which make up the toolbar. The image below represents the default settings for the Toolbar Options tab.


Image 1: Toolbar Preferences (Default)

Properties

Add > - will add selected item on left column to the right column, bringing this item to the toolbar.

< Remove - will take selected from right column to the left column, removing this item from the toolbar.

Add space - will add a space between the icons on your toolbar.

Reload Original - will return the left and right column and your toolbar to the original settings of McIDAS-V.

Export Selected to Plugin - will bring up the Plugin Creator window with the selected action included under the resources tab.

Export Selected to Menu Plugin - will bring up the Export to Menu window where the user can enter in a Menu Name and select which menu to add the action under.
