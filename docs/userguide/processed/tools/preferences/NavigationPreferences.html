User Preferences - Navigation Controls
The Navigation Controls tab allows you to change how McIDAS-V interprets mouse and keyboard events for navigating in 3D space. For each combination of mouse button and control/shift keys you can select an action.

Image 1: Navigation Controls (Default)

Note: The above image represents the default settings for the Navigational Controls' Mouse tab in the User Preferences Window.

The Cursor/Data Readout option brings up a latitude, longitude, and data value readout in the upper left part of the display on a mouse drag.

There are a number of navigation functions (e.g., rotate) that can be triggered by a key press. The Keyboard preference tab allows you to define the key and any modifier (i.e., control, shift):


Image 2: Keyboard Event Preferences (Default)

Note: The above Image represents the default settings for the Navigational Controls' Keyboard tab in the User Preferences Window.

When the Control box is checked next to a function, the Control key must be held down while pressing your defined key for the given function. When the Shift box is checked next to a function, the Shift key must be held down while pressing your default key for the given function.

 

Properties

None - The key combination on the mouse or keyboard with None selected will not perform any tasks.

Rotate - This allws the user to rotate the display. While most useful in the globe projection, it can be used at any time. If Rotate is followed by X, Y, or Z, that is the axis the display will rotate about. If there are no letters proceeding rotate, then the user can rotate the image about any axis. Negative and positive denote the direction that the image will rotate.

Zoom - This allows the user to zoom in to magnify a specific portion of the display, or zoom out to get a broader view.

Translate - This allows the user to shift the entire image in any direction with respect to the stationary background.

Selection - This allows the user to click and drag to make a box that the view screen will zoom in on.

Cursor/Data Readout - This allows the user to probe the data. Information of latitude/longitude and other variables plotted on the map will be displayed at the location your cursor is over the image.
