Data Sources Preferences

The User Preferences window is accessed in the Main Display window through the Edit->Preferences menu. The Data Sources tab allows the user to define what types of data choosers to use. This is a way to simplify the Data Explorer window by only showing the tabs that you want to use. The image below represents the default settings for the Toolbar Options tab.


Image 1: Data Sources Preferences (Default)

Properties
  • Use all data sources - Selects all data sources.
  • Use selected data sources - Uses only selected data sources.
  • - Selects all data sources.
  • - Unselects all data sources.
