User Preferences - Local Data

Image 1: Local Data Preferences
The Local Data tab allows you to add and manage local image files on your machine. To access the local datasets, select "

X Click to delete the local server entry.

Group Enter in a dataset name for the local server entry.

Descriptor Enter in a descriptor name for the new local server entry. This will appear under the Image Type pull down menu in the Satellite Imagery chooser.

Format Select an image format from the dropdown list.

File Mask Select the location of the local files using the pop up navigation box. At this time, only a folder can be selected to create a file mask.

Add new entry Click to create a new local server entry to enter in the group, descriptor, format, and file mask information.

Port information The port that the local server is listening on is listed below the Add new entry button.
