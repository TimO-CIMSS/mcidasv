5.3.2 Profiler 3D Multi-station Control
Overview
The Profiler 3D View shows winds at all levels above one or more stations in the main 3D view window. Winds are shown with the meteorological wind barb symbol.

Image 1: Profiler 3D Multi-station Control

Properties
  • Color Table

    The control has a color bar showing the active color table and the associated high and low data values in the units the display is made in. As the mouse pointer is moved over the color bar, the value at a particular color is shown. Click on the color bar to start the Color Table Editor. Or click on the button that displays the name of the color table to show a popup menu that allows you to change the range, select other color tables, etc.

  • Windbarb size

    Use the Windbarb size box to enlarge or reduce the wind barbs.

  • Vertical interval

    Use the Vertical interval box to change the vertical separation between wind barbs.
