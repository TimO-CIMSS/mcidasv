Observation List Controls
Overview

The Observation List Control shows a set of observation data in a textual list format.

The Select Fields button opens this Field Selector window:

Properties
  • - Opens a Field Selector window that allows you to select what Fields are listed in the table. To add a Field to the table, click on a Field under All Fields, and select Add and Apply. To remove a Field from the table, click on a Field under Current Fields, and select Remove and Apply.
  • Only Show Every - Allows you to set a time interval where data will be listed only once every N minutes. This makes the list in the chart shorter and easier to follow if having every time step is not important.
  • Show Raw Data - Displays the data in its raw format in the chart before any rounding/formatting is done to the data by McIDAS-V.
Menus

Many of the menu items seen utilizing this display are standard options that can be found in the Menus section of the Layer Controls Overview page. However, there is one option that is unique to this display.

The File menu has this unique option:

  • Export Table to CSV... - Allows you to save the data in the Comma-Separated Values (*.csv) format.
