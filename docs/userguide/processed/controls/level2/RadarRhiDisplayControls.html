Radar RHI Display Controls
Overview

The RHI (range-height indicator) display appears in the main 3D window and as a 2D plot in its display control. Level II Radar data has three moments or data types: Reflectivity, Radial Velocity, and Spectrum Width, indicated by the icon in the Field Selector. In order for Radar Cross Section to be listed as a valid Display Type in the Field Selector, either Reflectivity, RadialVelocity, or SpectrumWidth must be chosen, not one of their individual elevation angles. The control window contains two tabs, Display and Settings.

The Settings tab shows settings to control the display:

Properties
  • Station - Shows the station location id, latitude, longitude, and altitude.
  • Color Table - Shows the active color table and the associated high and low data values in the units the display is made in. As the mouse pointer is moved over the color bar, the value at a particular color is shown. Right click on the color bar or click on the button that displays the name of the Color Table to make modifications to the color bar. This allows you to open the Color Table Editor, change the range, select other color tables, etc.
  • Selector Position - Uses the slider to change the position of the selector in the main display. Alternatively, a numerical value can be entered to the right of the slider.
  • Vertical Scale - Changes the vertical axis range of the RHI line.
  • Autorotate RHI - Starts and stops autorotation of the RHI in the main 3D window. The 2D plot changes with the 3D RHI display when autorotate is on.
  • Selector Line Color - Selects the color of the selector line controlling the RHI azimuth in the main 3D window from the Edit->Select Line Color menu.
Menus

Many of the menu items seen utilizing this display are standard options that can be found in the Menus section of the Layer Control's Overview page. However, there is one option that is unique to this display.

The View menu has this unique option:

  • RHI - Offers options to capture an image or movie and to change many Display Window preferences.
