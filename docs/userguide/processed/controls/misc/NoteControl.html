5.7.2 Text Notes
Overview
McIDAS-V has a provision for you to record text notes, either stand-alone or associated with a display. These notes can be saved in Bundles. Bundles are small files of information that allow McIDAS-V to recreate a previous state of data and displays.

To create a stand-alone text note choose from the main menu Displays->Add text note. A window opens where you compose the note:

After you have finished with the note and have McIDAS-V in the state you want to show in future, save a named bundle file.

You can use a bundle for your own future reference, and you can provide this file to colleagues and they can use it to recreate the McIDAS-V state you saved. The text note you saved with the bundle will appear on their screen when McIDAS-V is started.

You can also attach a note to any or all individual parameter displays through the properties window: Edit->Properties. The text area will be opened in the bottom of the window in which you can compose your note.
