5.4.2 Vertical Profile Control
Overview

The Vertical Profile display control shows a plot of any number of field values versus altitude. The point that is being probed is shown in the main display. You can drag that point around to resample different areas.

You can add new fields to show with the Edit->Add Parameter... menu. See here for more information about customizing the chart.

Properties
  • Time Animation Control


    • controls looping of displays when more than one data time is loaded. See more in Time Animation Control.
