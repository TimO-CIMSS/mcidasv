Vertical Profile Controls
Overview

The Vertical Profile display controls show a plot of any number of field values versus altitude. You can add new fields to show with the Edit->Add Parameter... menu. See the Charts page for more information about customizing the chart.

Properties
  • Position - Displays the map position of the vertical profile is indicated by the "Lat: Lon:" readout below the graph. Enter in a value in the latitude and/or longitude field and press return to set the value to move the location of the probe.

    The map position of the profile is shown in the main 3D view window by a colored vertical line. There is a Selector Point on top of the vertical line. You can move the vertical profile line over the map by dragging the Selector Point with the left mouse button.

  • Time Animation Control - These controls start and stop looping of displays when more than one data time is loaded. See more in Time Animation Control.
  • Reset Probe Position - Resets the location of the probe to the default location of the center of the data boundaries. This option is found under the Edit menu.
  • Selector Color - Changes the color of the selector point on top of the vertical profile line by using the Edit->Selector Color menu.
  • Add Parameter... - Opens a Field Selector window with the option to add another paramater to display. This is accessed through the Edit->Add Parameter menu.
  • Probe Position - Controls how you can reposition the probe by preventing movement in the direction of certain axes.This is accessed through the View->Probe->Position menu. There is also an option to reset the probe back to its initial position.
  • Probe Size - Increases and decreases the size of the probe in the Main Display Window. This is accessed through the View->Probe->Size menu.
  • Probe Shape - Changes the shape of the probe in the Main Display Window. This is accessed through the View->Probe->Probe Shape menu.
