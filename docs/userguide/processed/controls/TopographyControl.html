5.5.1 Topography Control
Overview
The Topography control allows you to display topography datasets. The above image shows the control user interface which allows you to change the color table, shading and display mode.

The following image shows the actual display:

Properties
  • Color Table

    The control has a color bar showing the active color table and the associated high and low data values in the units the display is made in. As the mouse pointer is moved over the color bar, the value at a particular color is shown. Click on the color bar to start the Color Table Editor. Or click on the button that displays the name of the color table to show a popup menu that allows you to change the range, select other color tables, etc.

  • Shading

    Turning Shading off will provide for a blocky effect.

  • Display Mode

    There are 3 Display modes: solid, mesh and points, each providing a different type of display.
