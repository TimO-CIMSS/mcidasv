Building McIDAS-V from Source
Developers can download the source files for McIDAS-V and build it from scratch.

Before building McIDAS-V you need to first install Java and Java 3D on your system. You need to use the Java SDK, because the Java RunTime Environment (JRE) does not contain the Java compiler (javac). You need to use a version of Java 1.5 or above. You can use either the Java 3D SDK or JRE, version 1.3.x.

You will also need to install Ant 1.7 (http://ant.apache.org/) on your machine. Ant is similar to make - instead of a Makefile, Ant uses a build.xml file. Be sure to follow the instructions for installing Ant, especially those for setting up environment variables.

 

Download the following files:
  • The McIDAS-V source files (McIDAS-V_[version]_source.jar) from the McIDAS Website.
  • The IDV jar files (IDV_jars_[version].zip) from the Unidata website.

Set up the directory tree in the following structure:

		base_dir/
		base_dir/mcidasv/		-- Extract McIDAS-V source here
		base_dir/idv/
		base_dir/idv/lib/		-- Extract IDV jars here

The Ant build script "build.xml" will be extracted into "base_dir/mcidasv/"


Run "ant jar" to create "base_dir/mcidasv/dist/mcidasv.jar"

Run "ant run" to create mcidasv.jar and run McIDAS-V with a small java heap size for testing.
