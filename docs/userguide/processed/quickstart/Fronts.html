Displaying Fronts

This section describes how to make plots from the NOAA National Profiler Network.

In the Data Sources tab of the Data Explorer select Front Positions. For more information about the fronts chooser, see Choosing Front Positions.

In the Server: and Dataset: entry boxes, use the pull down list to select a remote server and dataset with front data, such as adde.ucar.edu and RTPTSRC and connect to the server.

Select either analysis or forecast fronts and click the button. The fronts will automatically plot in the Main Display Window.
