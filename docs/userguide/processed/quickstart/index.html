Getting Started

This section describes how to quickly get started using McIDAS-V and making displays of common data available.

McIDAS-V Main Windows

There are two main windows in the McIDAS-V application, the Data Explorer Window and the Main Display window. Other windows may appear when needed.

Data Explorer

The Data Explorer window is central to McIDAS-V. It is used to choose data sources and parameters to display, the kinds of displays to make, and times of data to display. More information can be found in the Data Explorer section of the McIDAS-V User's Guide.


Image 1: Data Explorer window

Main Display window

The Main Display window includes the McIDAS-V display panels, Legend, Time Animation Controls, view point controls for 3D displays, icons for zooming, panning, and rotating, menus of projections, and the main McIDAS-V menu. More information can be found in Main Display window section of the McIDAS-V User's Guide.


Image 2: Main Display window
