    • Display Changes

    Display Settings

    Users can save properties of display controls in favorite display settings. These can be used when displays are created or shared between controls. See the section on Display Settings in the User's Guide.

    Set vertical range in cross sections

    Users can now set the vertical scale range in the Cross Section and Data Transect Display controls.

    Point data accumulation

    Users can show the aggregation of point observations through time. This is useful for showing how many observations accumulate through time (e.g. earthquake epicenters, lightning plots) in a given area. For more information, see the Point Data Plot control documentation.

    Move a display to another view

    Users can now move a display from one view to another of the same type using drag and drop. Click on the drag and drop icon in the legend and drag it over to the new view. The display will be removed from the original view and placed in the new view. At present, only a move is allowed, not a copy. For more information, see the Display Control section of the IDV User's Guide.

    Data Changes

    New diagnostic functions

    The Jython library has been enhanced with many new functions. Most of the GEMPAK grid diagnostics have been implemented. For a complete list, see Jython Library Functions.

    GEMPAK grid reading

    The IDV can now read in GEMPAK grid files from local disk and from a THREDDS Data Server (TDS). Only grids with no packing, Grib 1 or Grib 2 packing are supported at this point.

    Saving grids locally

    You can now save remote grids to your local system and/or use the IDV to subset these remote grids. When you load in a gridded dataset, you can use the Write Grid menu from the Data Source properties menu to save the grid. The data will be saved in a netCDF file using the CF 1.0 convention. This same mechanism can be used to subset a local grid and/or convert from the native format to netCDF.

    Text Point Data

    Several improvements have been made to the Text Point Data Source. Point data stored in comma-separated value (CSV) text files or Excel spreadsheets can be read into the IDV. The data source has an metadata GUI which can be used to specify the parameters, the units and the structure of the file. For more information, see the Text Point Data File page in the User's Guide.

    Field Mask

    There is a new Field Mask option which allows you to define a mask for a dataset such that only certain fields are shown in the field selector. The field mask can be used for a particular dataset, or can be applied to similiar datasets. Access this feature through the Data Source properties menu using the Write Field Mask Plugin option.

    Other Changes

    Jython Shell

    A new interactive Jython Shell can be accessed under the Edit->Formulas->Jython Shell menu. This shell allows you to interactively write new procedures, load in data and create displays.

    Jython Library GUI

    The Jython library GUI has been updated to use a tree view instead of tabs to access the various libraries. It now has a search function and facilities for automatically creating formulas from your Jython methods. For more information, see the Jython Library documentation.
