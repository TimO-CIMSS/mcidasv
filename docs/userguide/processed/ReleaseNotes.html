Documentation Changes

Updated Content and Screenshots

The McIDAS-V User's Guide has been updated with all new screenshots and more detailed content.
Server Changes

Additional Local Servers

Additional local servers have been added to McIDAS-V to include other image servers besides AREA files. To set up the local servers, go to the Local Data Manager under the Tools menu.

Updated List of System Servers

The list of image system servers has been updated.
Other Changes

Changing Tab Names

A bug that affected the redrawing of tab names when changed has been fixed.

Manager Location Change

The Local Data Manager, formally found in the Edit->Preferences menu has been relocated to the Tools Menu. For more information, see the Local Data Manager page.

Skin Changes

Component Groups

The addition of the use of component groups has been implemented

Drag and Drop Tabs

Drag and drop tabs allow a user to reorganize the layout of tabs in a window, move tabs between windows, or take a tab and create a window with it.
Bundle Changes

Bundle Compatability with alpha9 and IDV

Bundles from alpha9 and prior, as well as from IDV now load into McIDAS-V.

Loading bug issues with alpha9

Previously, having the "Remove all data and displays" option checked when loading bundles in McIDAS-V would crash the main display window. This problem has been fixed.

New .mcv and .mcvz extensions

Bundles are now saved as .mcv (replaces .xidv) and .mcvz (.zidv) to distinguish a difference between bundles made in McIDAS-V and IDV.
Documentation Changes

Updated Content and Screenshots

The McIDAS-V User's Guide has been updated with all new screenshots and more detailed content.

Added McIDAS-X Bridge and HYDRA Documentation

New pages have been added to the Getting Started, Data Chooser, and Layer Controls sections of the McIDAS-V User's Guide about the McIDAS-X Bridge and HYDRA.
Server Changes

Local Servers

Local servers have been added to McIDAS-V. To set up the local servers, go to the Local Data tab in the preferences.
Other Changes

Alphabetized Data Sources List

The list of data sources users can select when loading data in the File or Directory choosers is not sorted alphabetically for an easier search.

Installer Changes

Mac OS X

The installer for Mac OS X has been updated to determine the OS version. Version 10.4 (tiger) uses JOGL and 10.5 (leopard) does not use JOGL.
Data Changes

Level II Remote

The remote Level II radar chooser has been updated due to changes on the remote server.
Other Changes

Changing Memory

To change the amount of memory used by McIDAS-V, edit the Maximum Heap Size in the Advanced tab of the Preferences by selecting Edit->Preferences... from the main menu. The new amount of memory will be saved and used in subsequent sessions. It is recommended to set this to no more than 1250 MB. The place where this information is stored was changed, so even if you have changed this value before, you will have to do it again. This probably will not change again in the future, but remember, this is an alpha!
