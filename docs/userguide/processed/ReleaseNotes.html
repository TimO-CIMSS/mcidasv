Release Notes
McIDAS-V
Version 1.0beta5
The items below reflect changes since the 1.0beta4 release.
Documentation Changes
  • Updated Content
Other Changes
  • Clock Added to Main Display Window
  • Scatter Analysis with Gridded Data
  • Fixes to Startup Manager
  • GOES-R Color Tables Added

Documentation Changes

Updated Content

The McIDAS-V User's Guide has been updated with more detailed content.
Other Changes

Clock Added to Main Display Window

A clock has been added to the Main Display window. To switch between the memory monitor and the clock, left click on the clock/monitor to switch back and forth.

Scatter Analysis with Gridded Data

The ability to create scatter diagrams was inadvertently omitted from the 1.0beta4 version of McIDAS-V. This function has been restored in 1.0beta5.

Fixes to Startup Manager

A fix has been made to address a problem with the startup manager if used the first time McIDAS-V is installed.

GOES-R Color Tables Added

Color tables to be used with GOES-R data have been added to McIDAS-V. For more information, see the Color Tables section.

Several of the Data Choosers have been updated for consistency.

A fix has been made to address the freezing problem that occurred when opening a new window on OSX.

McIDAS-V
Version 1.0beta2
The items below reflect changes since the 1.0beta1 release.
Documentation Changes
  • Updated Content
Chooser Changes
  • Updated Remote Soundings Chooser
  • New Grid Chooser Category
  • New Under Development Binary Chooser
Server Changes
  • Local Servers with spaces in directory path

Documentation Changes

Updated Content

The McIDAS-V User's Guide has been updated with more detailed content.
Chooser Changes

Updated Remote Soundings Chooser

The Remote Soundings choosers has been updated to remove hard coded values for UPPERMAND and UPPERSIG.

New Gridded Data Chooser Category

A new gridded data chooser category has been added to the list of choosers in the Data Explorer. For more information, see Choosing Gridded Data.

New Under Development Binary Chooser

A new binary chooser has been added to the Under Development chooser section. For more information see the Under Development section of the McIDAS-V forums.
Server Changes

Local Servers with spaces in directory path

A fix has been made to allow spaces in the directory path. To set up the local servers, go to the Local Data Manager under the Tools menu.

Absolute Times on Windows

A fix has been made which now allows absolute times to be used on Windows with local servers. To set up the local servers, go to the Local Data Manager under the Tools menu.
