    • Display Changes

    New Image Displays

    Two new image displays were added to this release for displaying satellite and NEXRAD Level III data - Image Display and Image Display over Topography. The old image display has been renamed to Image Sequence Display to reflect its specialty to that type of data. The Image Display will eventually replace the Image Sequence Display but the latter has been kept in for backward compatibility. It will be removed in a subsequent release.

    Chooser Changes

    Timeline Selector

    Several of the ADDE choosers now have a timeline for selecting absolute time intervals. For more information, see the User's Guide.

    Support for Multi-banded Images

    The Satellite Image Chooser now supports selecting all bands and/or all calibration types for multi-banded datasets. This simplifies loading data for formulas that need multiple channels of the image spectrum.

    Remote Level II Access

    You can install an experimental chooser for accessing NEXRAD Level II data from a THREDDS server. Use the Tools->Plugin Manager menu to install the plugin under the Miscellaneous section.

    Data Changes

    Parameter Groups Editor

    There is a new editor in the Tools menu for editing the resources which group parameters for use in system and user defined formulas. The Parameter Groups Editor allows users to define the specification of "parameter groups", named groups of parameter names. The parameter groups can be used the the Formula Editor to define the parameters that the formula needs by default.

    User Defined Derived Quantities

    McIDAS-V users can now make their formulas show up as automatically derived quantities in loaded data sources. For more information, see the Derived Quantities section of the User's Guide.

    Other Changes

    Animation Timeline

    Under the View->Animation Timeline menu, there is a new feature that allows you show the Time Animation for each data in the view in a graphical manner.

    New Installers

    There are two new installers available in the McIDAS-V Downloads page for Windows Vista and newer Linux systems. The Windows
    • Linux We have had to support an outdated version of Java 3D on Linux due to lagging support for the latest OpenGL versions. The new installer with the latest Java 3D should work with older OpenGL versions, although you'll get a warning when McIDAS-V starts up. This new version of Java 3D contains a significant performance improvement for point data displays.
    • Windows Vista The Windows Vista installer can be used on any Windows XP, 2000 or Vista system, but may give warnings on systems with older graphics hardware. Like the new Linux installer, it contains the performance improvements in the latest Java 3D.
