System Changes

Use of netCDF 4.0 library

McIDAS-V now uses the Java-netCDF 4.0 library. This provides limited support for HDF-4 and HDF-5 files.

Use of Java 1.6

McIDAS-V installers now use Java 6. Web Start still uses Java 5 (JDK 1.5) since there is no version of Java 6 on Mac OS-X for older systems.

Documentation Changes

Updated Content and Screenshots

The McIDAS-V User's Guide has been updated with all new screenshots and more detailed content.
Chooser Changes

Updated Layout

Many of the chooser have been updated for consistency of layout across all choosers.

Updated Usability with Server Manager

The data choosers and server manager have been updated for increased functionality and interaction while adding and manager servers.
Data Changes

Increased HYDRA functionality

New HYDRA displays, including Linear Combinations and 4 Channel Combinations have been added.

Addition of Scatter Analysis

A scatter analysis display has been added to McIDAS-V, allowing a user to create scatter diagrams of similar data types.

Objective Analysis of Point Data

Point data (observations) can now be gridded using a Barnes objective analysis routine. Numeric fields are listed under a "Gridded Data" category in the Field Selector. Users can control the analysis parameters through the Data Source properties or through the Data Subset panel. This work was made possible by Unidata using code from the Australian Bureau of Meteorology.

Remote Level III Radar access

The chooser for the Level II data from the THREDDS Data Server (TDS) can now be used to access Level III products as well. This provides faster access than the ADDE server and allows you to access data across days.

Weather Text Bulletins

Users can now access weather text products from a remote server or their own local archives. The server products use the ADDE protocol. The local archives are based on the GEMPAK NWX tables. For the latter, you must have GEMPAK installed and have the appropriate environment variables defined for the NWX program.

ATCF Tropical Storm track display

Users can access the Automated Tropical Cyclone Forecasting (ATCF) data from the National Hurricane Center in McIDAS-V. This archive has historical storm tracks back to the 1800's. Use the Data->Load Data->ATCF Storm Tracks menu to access this feature. The resulting display allows you to compare model forecasts in the database.

Server Changes

Additional Local Servers

Additional local servers have been added to McIDAS-V to include other image servers besides AREA files. To set up the local servers, go to the Local Data Manager under the Tools menu.

Updated List of System Servers

The list of image system servers has been updated.
Other Changes

Changing Tab Names

A bug that affected the redrawing of tab names when changed has been fixed.

Manager Location Change

The Local Data Manager, formally found in the Edit->Preferences menu has been relocated to the Tools Menu. For more information, see the Local Data Manager page.
