    The main toolbar has been enhanced with many new icons, for better representation of the functions available. Additionally, each button has it's own individual icon, as all instances of duplicate icons have been removed.

    Data Changes

    HYDRA

    HYDRA functionality has been increased to support several new data types, as well as increased functionality.

    Other Changes

    OS X Leopard

    McIDAS-V has been improved to have better behavior with Mac OS X Leopard.

    User's Guide

    The User's Guide has been improved (but not yet completed) with the addition of new and updated screenshots and content.
