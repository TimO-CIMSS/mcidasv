Drag and Drop Tabs
McIDAS-V has drag and drop tabs which allow you to organize multiple tabs and windows.

Image 1: Main Display Window with drag and drop tabs.

Drag and drop tabs allow you to separate tabs into individual windows, combine windows into tabs, as well as reorganzie the layout of tabs in a window. Drag tabs into separate windows by using the left click+drag option to drag the tab outside of the main display window. When the last tab in a window is moved into another window, the window will disappear. When tabs are rearranged or added from a different window, green arrows will point to the location the new tab will be placed.

Rename or close the tab by right clicking on it. Closing a tab will remove all layers associated with the display, but will not remove the data sources.
