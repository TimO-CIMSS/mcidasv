4.0.0 File Menu
  • New
    • This sub-menu allows you to create new windows.
    • View Window
      • Allows you to create new view windows.
    • Data Source
      • Allows you to open the Data Source Chooser window.
  • Open...
    • Allows you to select a previously created bundle to load in from your file system.
  • Save As...
    • Allows you to save the current application state as a bundle or Web Start JNLP file.
  • Save As Favorite...
    • Will save the state off as a specially located and named favorite. This allows you to easily create a set of favorite bundles and readily access them through the Displays->Favorite Bundles menu.
  • Default Bundle
    • This menu allows you to manage the "default bundle", the bundle that is loaded every time McIDAS-V starts up. See here for more information about bundles.
    • Save
      • Save the current application state as the default bundle.
    • Remove
      • Delete the default bundle.
    • Open
      • Load in the default bundle.
  • History
    • This menu lists the recently accessed data sources and allows you to easily load them.
  • Exit
    • Exits the application.
4.0.1 Edit Menu
  • Remove All Displays
    • Remove all displays.
  • Remove All Data Sources
    • Remove all loaded data sources. The displays are unaffected.
  • Remove All Displays and Data
    • Removes all displays and loaded data sources.
  • Formulas
    • This menu contains items related to end user formulas.
    • Create Formula
      • Show the Formula dialog window to create a new formula.
    • Jython library
      • Show the Jython library editor.
    • Import
      • Import a formula from a file.
    • Export
      • Export a formula to a file.
    • Edit Formulas
      • Edit a previously defined formula.
  • Delete Saved Views
    • This menu allows you to remove the named saved views (created with the Views->Saved Views->Save View State menu).
  • Preferences
    • Show the User Preferences window.
4.0.2 Displays Menu
The displays menu contains facilities to create new displays and lists the currently created displays.
  • Favorite Bundles
    • Bundles hold application state. Favorite bundles are just normal bundles that are categorized and saved off in a particular location on disk and serve as a convenience to the user. This allows the user organize their favorites by category and readily load them in.
    • Save As Favorite
      • Save the current McIDAS-V state as a favorite bundle. This will show a dialog that prompts you for a category and bundle name.
    • Manage
      • This shows the list of favorites that you have created and allows you to manage the categories and remove favorite bundles.
    • Rest of Menu List
      • The rest of this menu holds menu items for the favorite bundles, categorized by the favorites category.
  • Display Templates
    • Provides ability to access and manage the saved display templates
  • Maps and Backgrounds
    • This menu provides facilities for creating new map displays or background image. Note, once a map display is created you can add new individual maps to that particular display.
    • Add Your Own Map
      • Select a shape file or McIDAS map file from the local disk.
    • Add Default Maps
      • The default map is the map display (the set of actual maps, their color, line width, etc.) that is automatically created in each view window. You can define what is the default map through an instance of a map display control. This menu item creates a new map display control with the default map state.
    • Add System Map
      • McIDAS-V comes pre-configured with a set of base system maps. This menu allows you to create new map displays using these system maps.
    • Add Background Image
      • Select a background image for the display using the Background Image Control.
  • Locations
    • This menu lists a variety of point location plots that can be created: NEXRAD sites, state names, state locations, etc. It will create a Location Control with the chosen location information.
  • Special
    • This menu lists allows you to create displays that do not require data.
    • Range Rings
      • Create a radar range ring display. Radar range rings are a set of concentric rings centered on a point, and radial lines, used to show distance and bearing from the center. The initial position is at the center of the view's current projection. For more information see Range Rings. You do not need to use radar data to see this measuring device.
    • Range and Bearing
      • Creates a display to show the range and bearing between any two points. The initial position is near the center of the current entire map area. If you are zoomed in, the range and bearing line may be off screen. For more see Range and Bearing Line.
    • Text note
      • Creates a simple text message box. This note can be saved for later use, or sent to other users of McIDAS-V so they can see the same display with your notations. See Text Notes.
    • Drawing Control
      • Create a drawing tool that allows you to annotate McIDAS-V displays.
    • Transect Drawing Control
      • Create a drawing tool that allows you to create transects for the Transect View
    • Location Indicator
      • Creates a location indicator cross-hair on the display. See the Location Indicator Control.
    • WebCam Display
      • Creates a display of available WebCam sites. See the WebCam Control.
    • WorldWind Locations
      • Add locations from the WorldWind database. See the Location Control
  • Current View
    • This is essentially the View and Projections menu of the currently active view window.
  • Current Displays
    • If displays have been added to McIDAS-V, they are shown at the bottom of the Displays menu. Selecting one of these items brings up that displays control window.
4.0.3 Data Menu
This menu allows you to load in a New Data Source from the Data Source Chooser. It also lists all of the currently loaded formulas and data sources and provides options to select fields and edit properties.
  • New Data Source
    • Provides ability to access the individual Data Source Chooser-s from the sub menus.
  • Favorite Data Sources
    • Provides ability to access and manage the saved favorite data sources
  • Formulas
    • Provides ability to access formulas
4.0.4 Tools Menu
This menu allows you to configure and edit different McIDAS-V resources.
  • Color Table Editor
    • Show the Color Table Editor, which allows you to view, create and modify color tables.
  • Station Model Editor
    • Show the Station Model Editor, which allows you to view, create and modify station models for plotting point data.
  • Projections Manager
    • Show the Projection Manager, which allows you to choose, edit and save map projections for use in Unidata applications.
  • Parameter Alias Editor
    • Show the Parameter Alias Editor, which allows you to define aliases for parameters in data sets that McIDAS-V does not know about.
  • Parameter Defaults Editor
    • Show the Parameter Defaults Editor, which allows you to define the default display conventions (e.g., color tables, ranges, etc.) for parameters.
  • Parameter Groups Editor
    • Show the Parameter Groups Editor, which allows you to define the groups of parameters for use in derived data formulas.
  • Plugin Manager
    • This menu allows you to manage McIDAS-V plugins.
  • Plugin Creator
    • This menu allows you to create plugins.

4.0.6 View Menu
Each display or View Manager has a View and Projections menu. The View menu lists the displays shown in this window, change the point of view, capture images and movies, and make some other changes in the overall view.
  • Displays
    • This menu deals with the displays that are shown in this particular view, which are shown at the bottom of the menu.
    • Legend
      • Show the display legend in their own window.
    • Remove All Displays
      • Remove the displays shown in this view.
    • Visibility Animation
      • Cycle through the visibility of each display, showing one at a time. Click the check box "on" to start animation, check off to stop. You can increase or decrease the speed with the provided Faster and Slower choices. Note this is not a time animation loop; it loops through the visibility of the displays present in this view window.
  • Capture
    • You can capture the current view as a JPEG image, send it to a printer, or capture a QuickTime movie. See here for more information.
  • Show
    • Different Flags
  • Viewpoint
    • This menu allows you to change the point of view.
    • Top View through West
      • These 5 items reset the point of view to one of the orthogonal directions.
    • Perspective View
      • Provides an alternate 3D view of the display. The default is a parallel projection.
    • Viewpoint Dialog
      • This shows a dialog box for you to enter the azimuth and tilt of a desired point of view. Azimuth is from North, clockwise to 360 degrees. Tilt is degrees down from overhead, vertical. A top view has tilt of 0. Oblique views from the southeast have azimuth 135, and a tilt of about 45 degrees. A view from the "east" has an azimuth of 90, tilt 90.
    • Vertical Scale
      • This shows a dialog box to set the vertical scaling between the top and bottom of the wire frame box. The scale is linear. You can specify the units for the max and min values. Units must be convertible with meters (e.g., km, ft, fathoms, etc)
  • Color
    • Change the background and foreground colors of the view (not the map lines) to predefined colors or set your own. When setting your own colors you can also save off the chosen color pair as a named predefined color pair which will appear in the list.
  • Saved State
    • This menu allows you to save the current view state (e.g., projection, colors, viewpoint) and to load previously saved view states into the current view window.
  • Full Screen
    • Show the current view in full screen mode.
  • Animation Timeline
    • Show the Animation Timeline
  • Properties
    • Show the properties dialog for the view. Allows you to set aspect ratio, vertical scale, flags, color, etc.
4.0.7 Projections Menu
The Projections menu lets you change the map projection in the 3 dimensional display.
  • Predefined
    • Contains the list of pre-defined projections you can choose from, including both system-supplied and your own projections.
  • From Displays
    • Lists the native data projections of any displays being show.
  • History
    • Contains the projections that have been used for this view. It allows you to easily go back to prior projections.
  • New/Edit
    • Show the Projection Manager dialog to view, create and edit projections.
  • Use Displayed Area
    • Resets the projection to that of the current zoom level.
  • Go to Address
    • Change the projection to a given address, zip code, city, etc.
    The formats include: 
    12345 oak street, my town, my state
my town, my state
80303 (zip code)
latitude longitude
"ip" for the location of this computer
  • Auto-Set Projection
    • Toggles the behavior of automatically setting the projection to the projection of the most recent display that has been added.
  • Share Views
    • Control whether Sharing is turned on or off for this view. If sharing is turned on then view point and projection changes are shared with any other views that also have sharing turned on.
