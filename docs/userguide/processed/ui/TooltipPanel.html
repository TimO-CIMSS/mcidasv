Cursor Readout and Tooltip Panel

At the bottom of the Main Display window is a cursor readout/tooltip panel. This panel is useful for seeing how much memory has been used, the current time, as well as the value of the parameter probed in the Main Display window. You can change the panel back and forth between displaying memory or time by left-clicking on the memory or time that is currently being displayed. Latitude, Longitude, and Altitude values are always displayed, regardless of if memory or time is being displayed in the panel. These values are listed for the location of your mouse of the the Main Display window.

Memory

Image 1: Memory Readout

You can change the amount of memory allocated by McIDAS-V at startup in the Advanced tab of the User Preferences window. When McIDAS-V has used up the majority of its available memory, the memory section of the cursor readout/tooltip panel will turn pink and then red. This will cause McIDAS-V to slow down, and may eventually prevent new data sources from being loaded in. You can solve this problem by right clicking on the memory and selecting: . This will clear out some memory so you can still work efficiently in your current session.

Time

Image 2: Time Readout

The time displayed in the cursor readout/tooltip panel is not the time associated with the data you have loaded into McIDAS-V, but rather the current time in GMT.

Probe Readout

Image 3: Probe Value Readout

As a default in the Navigation Controls tab of the User Preferences window, holding down the middle mouse button over your display will probe the data and give a numerical readout of your displayed parameter(s). The value(s) will be listed at the bottom of the Main Display window, in the Probe Readout panel. This panel lists the location of your mouse when you probe the data with respect to Latitude and Longitude. Also, as a default, this panel will list your data source, the display type, and the numerical value of your parameter(s) at the location of your mouse with units if they are included in your data source. You can change how your data source will be labeled in this panel by right clicking on the item in the Legend, and going to Edit->Properties.... In the Settings tab of this Properties dialog, you can change the Legend Label to say anything, including adding macros to further customize the readout in this panel.
