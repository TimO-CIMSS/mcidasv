Displays
The Displays menu lists the display types applicable to the selected Field. This is the display type that will be used to display your data in the Main Display Window.

Creating Displays

The Display menu shows a hierarchical list of the available displays for a particular field. Click on the type of display you want from the choices listed in the Displays menu. There are a variety of different display types that are unique to particular types of data. For more information, see the Examples of Display Types pages.

You can select two or more displays (different display types) at one time. For an additional display, hold down the Control key and click on the second display type desired. Then go ahead and create the display as normal; all selected displays will be made. To make the display click or double click on the display.
