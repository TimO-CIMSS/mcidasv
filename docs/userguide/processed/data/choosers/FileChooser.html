Choose Data on Disk

The General->Files chooser allows you to select data from your file system.


Image 1: File Chooser

To select files:

  1. Navigate to the directory that contains the files you are interested in. The Look In menu allows you to select directories or you can double-click on directory names. You can use the Directory History widget to navigate to frequently used directories. This list contains the last 15 directories that data was loaded from.
  2. Select a file or files. Multiple files can be selected using Control-Click or Shift-Click.
  3. Optionally, specify the type of data the file contains. Normally McIDAS-V figures out what kind of data is in a file by the filename itself. You can override this using the Data Source Type menu.
  4. The Update button causes a rescan of the current directory. Use this to update the file chooser if the files have changed.
  5. When done, load the selected file with the Add Source button.
  6. The data file selected appears in the Field Selector window.
