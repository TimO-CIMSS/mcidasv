3.4.1 Chooser a URL
The URLs chooser allows you to specify the internet location (URL) of a data source. This URL may be a web page, a bundle or any data file that McIDAS-V can process from a URL.

Image 1: URL Chooser
To select a URL:
  1. Enter the URL into the field or choose a previously selected URL from the pull down menu.
  2. You can enter multiple URLS, one per line, by toggling the button.
  3. Optionally, specify the type of data the URL contains. Normally McIDAS-V figures out what kind of data is in a file by the URL itself. You can override this using the Data Source Type menu.
  4. When done, load the selected URL with the Add Source button.
