3.4.8 Choose NOAA National Profiler Network Data
The Profiler chooser allows you to access NOAA National Profiler Network data on ADDE servers.

Image 1: Profiler Chooser
To select Profiler data you:
  1. Select an ADDE server and data interval and press Connect. The data interval sets the time intervals in the display; NOAA profiler data is available in 6 minute or one hour intervals.
  2. Once connected the available times will be shown. By default the last 12 times will be selected. Times can be selected in the list or the time line (use control/click or shift/click in the list to select multiple times). See the Timeline section for details on its use.
  3. Select one or more profiler stations. See the station map section for more information on how to use the station map.
  4. When done, load the selected profiler data with the Add Source button.
