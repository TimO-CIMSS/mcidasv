Choose RAOB Data (atmospheric balloon sounding)

You can access RAOB data either from remote ADDE servers (using the Point Observations->Upper Air->Soundings->Remote chooser, pictured below) or from local files (using the Point Observations->Upper Air->Soundings->Local chooser). The only difference between these two choosers is specifying the source of data. You either select an ADDE server and press Connect or you select a file containing RAOB data.


Image 1: RAOB Chooser

To select RAOB data you:

  1. Choose the server or the local file. The list of times will be displayed. In this chooser you select particular stations at particular times.
  2. Select a set of dates and times that you want to use. To select more than one time use Control-Click or Shift-Click.
  3. Select one or more stations for the given time using the map. See below for more information on how to use the station map.
  4. Once selected the station/time combination will be shown in the Selected Stations.
  5. When done, load the selected RAOB data with the Add Source button.
Choose Single Level RAOB Data

RAOB data can also be selected (from an ADDE server) as a single level from the Point Observations->Upper Air->Plot chooser. This data can then be displayed as a station model.


Image 2: Single Level RAOB Chooser

To select single level RAOB data you:

  1. Select an ADDE server and data type and press Connect.
  2. Either, choose a relative set of times steps (e.g., Last 5 times).
  3. Or, choose a set of absolute times (e.g., 12Z, 1Z); When choosing absolute times for the first time McIDAS-V needs to query the ADDE server for the times. This may take some time. Times can be selected in the list or the time line (use control/click or shift/click in the list to select multiple times). See the Timeline section for details on its use.
  4. Select the level you want to view.
  5. Optionally select the station model to be used for displaying this data.
  6. When done, load the selected RAOB data with the Add Source button.
