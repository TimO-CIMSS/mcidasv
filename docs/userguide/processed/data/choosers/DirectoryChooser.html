Polling on files in a Directory

The General->Directory chooser allows you to select files in a directory by time. You can select the most recent N files or select all files updated in a specific time period. For more information on using this chooser, see Getting Started.


Image 1: Directory Chooser

Properties
  • Data Type - If left as "I'm Feeling Lucky", McIDAS-V will figure out what kind of data is in the file by the filename itself. This can be overridden by selecting the type of data the file contains from the pull down menu. Click here for a list of available data source types.
  • Source Name - Enter in a name for the dataset to be shown in the Field Selector.
  • Directory - Enter in a folder location or click Select to bring up a dialog box to locate a folder.
  • Recent Files - Choose the number of files you want to search for by an absolute number, or by the files that have been updated in a certain number of minutes.
  • Refresh - When the checkbox is selected, the files and associated displays will automatically update. Change the number of minutes to control the update frequency.
  • File Pattern - You can choose to enter in a patter for the files you want to poll on.
  • Verify - Click to verify the files that will be scanned.
  • Include Hidden Files - When clicked, hidden files will be inclucded in the polling.
  • Help - Brings up this help page.
  • Refresh - Updates the directory chooser with the most recent files.
  • - Loads the selected files.
