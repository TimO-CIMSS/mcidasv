Poll on files in a Directory

The General->Directory chooser allows you to select files in a directory by time. You can select the most recent N files or select all files updated in a specific time period.


Image 1: Directory Chooser

To select data from a directory you:

  1. Specify the type of data the file contains. Normally McIDAS-V figures out what kind of data is in a file by the filename itself. You can override this using the Data Source Type menu.
  2. Select the directory you want to poll on.
  3. Type in a name for the dataset that you want to show in the Field Selector.
  4. Optionally, set the pattern for the files you want to poll on. Use the check box to set whether you want to poll on hidden files. You can verify the files that will be scanned using the pattern by pressing the Verify button.
  5. Choose the number of files you want to search for. You can select an absolute number of files or you can select the files that have been updated in a certain time period.
  6. If you want to have the files and associated displays automatically update, set the Polling information and the update frequency.
  7. When done, load the selected data with the Add Source button.
