Choose Point Observation Data

The Point Observations->Surface Plot choosers allow you to choose global hourly surface weather station observations (METAR), and Synoptic data from remote ADDE servers.


Image 1: Point Data Chooser

To select Point data:

  1. Specify an ADDE server and data type and press connect. This chooser works similarly to the Image chooser described above. Once connected to a server the list of available times will be shown.
  2. Choose a relative set of times steps (e.g., Last 5 times). When choosing relative times you can set the time increment to use, e.g., Using "3 most recent" times with a "3"hour time increment would result in the most recent time, one 3 from 3 hours ago and one from 6 hours ago.
  3. Or, choose a set of absolute times (e.g., 12Z, 1Z); When choosing absolute times for the first time McIDAS-V needs to query the ADDE server for the times. This may take some time. Times can be selected in the list or the time line (use control/click or shift/click in the list to select multiple times). See the Timeline section for details on its use.
  4. Optionally, select the station model you want to use when displaying this data as a point data plot.
  5. When done, load the selected point data with the Add Source button.
