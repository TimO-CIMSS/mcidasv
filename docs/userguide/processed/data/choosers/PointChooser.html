Choosing Point Observation Data

The Point Observations->Surface Plot choosers allow you to choose global hourly surface weather station observations (METAR), and Synoptic data from remote ADDE servers. For more information on how to use this chooser, see Getting Started.


Image 1: Point Data Chooser
Properties
  • Server:
  • Enter in a server name or choose one from the pull down menu. Right click in the entry box to delete the server from the list.
  • Data Type:
  • Enter in a dataset name or choose one from the pull down menu. Right click in the entry box to delete the dataset from the list.
  • Connect
  • Click to connect to and query the server for available times.
  • Times: Relative
  • Allows you to select a group of up to the last 50 times.
    • Relative Time Increment:
    • Set a time increment to use with relative times. For example, using "3 most recent" times with a "3" hour time increment would result in the most recent time, one from 3 hours ago and one from 6 hours ago.
  • Times: Absolute
  • Allows you to choose one or more absolute times. When choosing absolute times for the first time McIDAS-V needs to query the ADDE server for the times. This may take some time. To select more than one time use Control-Click or Shift-Click. See the Timeline section for details on its use.
  • Layout Model:
  • Select the station model to use when displaying this data as a point data plot.
  • Add Source
  • Click to load the selected point data.
  • Update
  • Click to update the surface point chooser with the most recent data.
  • Help
  • Click to bring up this help page.
  • Create Display
  • When checked, the display will automatically be created.
