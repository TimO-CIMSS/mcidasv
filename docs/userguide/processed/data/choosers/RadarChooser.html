Choose NEXRAD Level III Radar Data

The Radar Imagery->Level III chooser allows you to choose NEXRAD Level III data on remote ADDE servers.


Image 1: Level III Radar Chooser

The Level III chooser works similarly to the Image chooser described above:

  1. Choose an ADDE Server and a Dataset to use:

  2. Press Connect:
    3. This causes McIDAS-V to query the server for available radar types and stations.

  3. Select a station using the map:

  4. Select a radar type:
    5. Once a station and a radar type has been selected the ADDE server will be queried as to the available times. Optionally, you can override the default data type that is returned.

  5. Or, choose a set of absolute times (e.g., 12Z, 1Z):
    6. When choosing absolute times for the first time McIDAS-V needs to query the ADDE server for the times. Times can be selected in the list or the time line (use control/click or shift/click in the list to select multiple times). See the Timeline section for details on its use.

  6. When done, load the selected radar data with the Add Source button.
