Choose Satellite Imagery

The Satellite Imagery chooser allows you to access satellite imagery on remote ADDE servers.


Image 1: Image Chooser

To select images:

  1. Choose an ADDE Server and a Dataset to use. Press Connect to query the server for available image types.
    2. McIDAS-V comes pre-configured with a list of ADDE servers and data sets or you can enter your own. See Available data for a description of these pre-defined data sets.

  2. Select the Image Type to use:

  3. Either, choose a relative set of times steps (e.g., Last 5 times):

  4. Or, choose a set of absolute times (e.g., 12Z, 1Z):
    5. When choosing absolute times for the first time McIDAS-V needs to query the ADDE server for the times. This may take some time. Times can be selected in the list or the time line (use control/click or shift/click in the list to select multiple times). See the Timeline section for details on its use.

  5. Optionally override the image selection properties (e.g., image size, location, etc.) See below for more details.

  6. When done, load the selected image data with the Add Source button. The Create display button, if selected will automatically create the image display after loading the data source.

Image Properties

The Properties section allows you to override the default properties McIDAS-V uses to specify other aspects of the image data. The widgets show the defaults McIDAS-V will use. By changing any of the widgets you can override the default value.

You can specify the data type that is to be returned (e.g., BRIGHTNESS, RAW) from the Data Type pull-down list. If the image has multiple channels you can also choose which channel to use.

You can specify the placement point of the image (e.g., Center, Upper Left) and its location. Location can either be specified as an Line/Element coordinate or as Latitude/Longitude. Use the cycle buttons to select the type of location to use.

Image size and magnification can also be set. Click on the slider icon to change the values. The lock icon to the right of the Image Size will lock the image size, but allow you to still be able to alter the magnification for the image.
